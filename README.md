# NAI Prompt Manager

A floating prompt manager userscript for NovelAI.

## Features
- 多工作空间分类管理
- 图片画廊 / 瀑布流 / 列表三种视图
- PNG 元数据自动识别（采样器/CFG/Seed/负面提示词等）
- 一键发送图片到 NAI 生成界面
- 卡片自定义背景色 / 双色渐变
- 10 个本地存档槽位
- ZIP 导入导出

## 安装
1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 点击 [这里](raw 链接) 安装脚本

## 使用
- 进入 https://novelai.net/ 后，右下角出现悬浮按钮
- 点击或按 `Shift + P` 打开主面板
